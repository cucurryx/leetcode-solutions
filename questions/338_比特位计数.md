# 338.比特位计数

## 问题

给定一个非负整数 **num**。对于 **0 ≤ i ≤ num** 范围中的每个数字 **i** ，计算其二进制数中的 1 的数目并将它们作为数组返回。

**示例 1:**

```
**输入:** 2
**输出:** [0,1,1]
```

**示例 2:**

```
**输入:** 5
**输出:** [0,1,1,2,1,2]
```

**进阶:**

* 给出时间复杂度为**O(n\*sizeof(integer))**的解答非常容易。但你可以在线性时间**O(n)**内用一趟扫描做到吗？
* 要求算法的空间复杂度为**O(n)**。
* 你能进一步完善解法吗？要求在C++或任何其他语言中不使用任何内置函数（如 C++ 中的 **\_\_builtin\_popcount**）来执行此操作。
