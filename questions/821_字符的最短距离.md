## 问题

**示例 1:**

```
**输入:** S = "loveleetcode", C = 'e'
**输出:** [3, 2, 1, 0, 1, 0, 0, 1, 2, 2, 1, 0]

```

**说明:**

1. 字符串 `S` 的长度范围为 `[1, 10000]`。
2. `C` 是一个单字符，且保证是字符串 `S` 里的字符。
3. `S` 和 `C` 中的所有字母均为小写字母。
