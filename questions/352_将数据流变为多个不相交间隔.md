# 352.将数据流变为多个不相交间隔

## 问题

给定一个非负整数的数据流输入 a<sub>1</sub>，a<sub>2</sub>，…，a<sub>n，</sub>…，将到目前为止看到的数字总结为不相交的间隔列表。

例如，假设数据流中的整数为 1，3，7，2，6，…，每次的总结为：

```
[1, 1]
[1, 1], [3, 3]
[1, 1], [3, 3], [7, 7]
[1, 3], [7, 7]
[1, 3], [6, 7]

```

**进阶：**  
如果有很多合并，并且与数据流的大小相比，不相交间隔的数量很小，该怎么办?
