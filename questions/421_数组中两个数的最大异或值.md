# 421.数组中两个数的最大异或值

## 问题

找到 a<sub>i</sub> 和a<sub>j&nbsp;</sub>最大的异或 (XOR) 运算结果，其中0 ≤ *i*, *j* < *n* 。

你能在O(*n*)的时间解决这个问题吗？

**示例:**

```
**输入:** [3, 10, 5, 25, 2, 8]

**输出:** 28

**解释:** 最大的结果是 **5** ^ **25** = 28.

```
