# 368.最大整除子集

## 问题

给出一个由**无重复的**正整数组成的集合，找出其中最大的整除子集，子集中任意一对 (S<sub>i，</sub>S<sub>j</sub>) 都要满足：S<sub>i</sub> % S<sub>j</sub> = 0 或 S<sub>j</sub> % S<sub>i</sub> = 0。

如果有多个目标子集，返回其中任何一个均可。

**示例 1:**

```
**输入:** [1,2,3]
**输出:** [1,2] (当然, [1,3] 也正确)

```

**示例 2:**

```
**输入:** [1,2,4,8]
**输出:** [1,2,4,8]

```
