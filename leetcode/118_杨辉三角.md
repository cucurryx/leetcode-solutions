# 118. 杨辉三角

## 问题

给定一个非负整数 numRows，生成杨辉三角的前 numRows 行。

在杨辉三角中，每个数是它左上方和右上方的数的和。

```
示例:

输入: 5
输出:
[
     [1],
    [1,1],
   [1,2,1],
  [1,3,3,1],
 [1,4,6,4,1]
]
```

## 解答
处理好边界就行了。
```C++
class Solution {
public:
    vector<vector<int>> generate(int numRows) {
        vector<vector<int>> res;
        for (int i = 0; i < numRows; ++i) {
            vector<int> vec(i+1, 1);
            if (i > 1) {
                for (int j = 1; j < i; ++j) {
                    vec[j] = res[i-1][j] + res[i-1][j-1];
                }
            }
            res.push_back(vec);
        }
        return res;
    }
};
```
